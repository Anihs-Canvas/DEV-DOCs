    1.1 Introduction to Terraform

    What is Terraform?

    🏗️
    Terraform Definition

    Terraform is HashiCorp's open-source Infrastructure as Code (IaC) tool that allows you to define and provision infrastructure using a declarative configuration language.

    Key Characteristics
    📝
    Declarative Infrastructure Management

    Define what you want, not how to create it. Terraform figures out the steps needed to reach your desired state.

    resource "aws_instance" "web" {
  ami           = "ami-0c55b159cbfafe1f0"
  instance_type = "t2.micro"
  
  tags = {
    Name = "WebServer"
  }
}

    HashiCorp's Open Source Tool

    Free, community-driven with enterprise options. Over 1000+ providers available.
