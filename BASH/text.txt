Essential AWS Services for Django, DevOps & Cloud Developers

=== COMPUTE ===
⦁	Amazon EC2 - Virtual servers for hosting applications
⦁	AWS Lambda - Serverless functions for event-driven code
⦁	AWS Elastic Beanstalk - Easy application deployment and management
⦁	AWS Fargate - Serverless containers

=== STORAGE ===
⦁	Amazon S3 - Object storage for static files, media, backups
⦁	Amazon EBS - Block storage for EC2 instances
⦁	Amazon EFS - Shared file system for multiple instances

=== DATABASE ===
⦁	Amazon RDS - Managed PostgreSQL/MySQL for Django
⦁	Amazon Aurora - High-performance PostgreSQL/MySQL compatible
⦁	Amazon DynamoDB - NoSQL for high-scale applications
⦁	Amazon ElastiCache - Redis/Memcached for caching

=== NETWORKING & CONTENT DELIVERY ===
⦁	Amazon VPC - Virtual private network for resources
⦁	Elastic Load Balancing (ALB/NLB) - Distribute traffic across instances
⦁	Route 53 - DNS management and domain routing
⦁	AWS CloudFront - CDN for static assets and faster content delivery

=== DEVELOPER TOOLS ===
⦁	AWS CodePipeline - CI/CD automation
⦁	AWS CodeBuild - Build and test code
⦁	AWS CodeDeploy - Automated deployments
⦁	AWS CodeCommit - Git repository hosting

=== MANAGEMENT & GOVERNANCE ===
⦁	AWS CloudWatch - Monitoring, logs, and alerts
⦁	AWS CloudFormation - Infrastructure as Code (IaC)
⦁	AWS Systems Manager - Server management and automation
⦁	AWS CloudTrail - Audit logs and API tracking

=== CONTAINER SERVICES ===
⦁	Amazon ECS - Container orchestration
⦁	Amazon EKS - Managed Kubernetes
⦁	Amazon ECR - Container image registry

=== SECURITY & IDENTITY ===
⦁	AWS IAM - User and permission management
⦁	AWS Cognito - User authentication for applications
⦁	AWS Certificate Manager (ACM) - Free SSL/TLS certificates
⦁	AWS Secrets Manager - Store database credentials and API keys
⦁	AWS KMS - Encryption key management

=== APPLICATION INTEGRATION ===
⦁	Amazon SQS - Message queuing for async tasks
⦁	Amazon SNS - Push notifications and pub/sub messaging
⦁	Amazon API Gateway - Create and manage REST/GraphQL APIs
⦁	AWS Step Functions - Workflow orchestration

=== ANALYTICS & MONITORING ===
⦁	Amazon CloudWatch Logs Insights - Query and analyze logs
⦁	AWS X-Ray - Distributed tracing and debugging
⦁	Amazon Athena - SQL queries on S3 data

=== BACKUP & DISASTER RECOVERY ===
⦁	AWS Backup - Centralized backup management
⦁	Amazon S3 Glacier - Long-term archival storage

=== TYPICAL DJANGO STACK ON AWS ===
1. EC2/ECS/Fargate - Run Django application
2. RDS (PostgreSQL) - Primary database
3. ElastiCache (Redis) - Session store, caching, Celery broker
4. S3 - Static files (CSS/JS) and media uploads
5. CloudFront - CDN for static/media files
6. ALB - Load balancing and SSL termination
7. Route 53 - Domain and DNS management
8. SQS - Celery task queue
9. CloudWatch - Application logs and monitoring
10. IAM - Access control and service permissions

=== DAILY DEVOPS TASKS ===
⦁	CloudWatch - Check logs and metrics
⦁	EC2/ECS - Monitor instance health
⦁	CodePipeline - Deploy code changes
⦁	IAM - Manage user access
⦁	RDS - Database performance tuning
⦁	S3 - Manage backups and static files
⦁	CloudFormation - Update infrastructure
⦁	Systems Manager - Execute commands on servers
