# Complete Celery Django Documentation

## Part 1: Introduction to Celery in Django

### What is Celery?

Celery is a distributed task queue system for Python that allows you to execute tasks asynchronously outside of the HTTP request-response cycle. It's particularly useful in Django applications for handling time-consuming operations that would otherwise block the web server and degrade user experience.

### Why Use Celery with Django?

1. **Asynchronous Processing**: Execute long-running tasks without blocking HTTP responses
2. **Scalability**: Distribute tasks across multiple workers and machines
3. **Reliability**: Tasks can be retried, scheduled, and monitored
4. **Performance**: Improves application responsiveness by offloading heavy operations
5. **Background Jobs**: Handle periodic tasks, email sending, data processing, etc.

### Core Concepts

**Task**: A unit of work that can be executed asynchronously
**Worker**: A process that executes tasks
**Broker**: Message transport system (Redis, RabbitMQ, etc.)
**Result Backend**: Storage for task results
**Beat**: Scheduler for periodic tasks

### Architecture Overview

```
Django App → Celery → Message Broker → Celery Workers → Result Backend
     ↓           ↓           ↓              ↓              ↓
  Web Request  Task Queue  Redis/RMQ   Task Execution   Task Results
```

### Use Cases in Job Portal Application

Using our job portal models, here are common Celery use cases:

1. **Email Notifications**: Send job alerts to users
2. **Data Processing**: Bulk import job postings
3. **Report Generation**: Create analytics reports
4. **Search Indexing**: Update search indexes when jobs are posted
5. **Cleanup Tasks**: Remove expired job postings
6. **API Integrations**: Sync with external job boards

## Part 2: Theory Generation - Celery CRUD Operations and Syntax

### 2.1 Task Definition and Registration

#### @app.task Decorator

**Full Description**: The `@app.task` decorator is the primary way to define a Celery task. It converts a regular Python function into a Celery task that can be executed asynchronously by workers. The decorator registers the function with the Celery application and provides various configuration options for task behavior.

**Syntax**:
```python
@app.task(bind=False, name=None, base=None, retry=True, max_retries=3, default_retry_delay=60)
def task_function(arg1, arg2, kwarg1=None):
    # Task logic here
    return result
```

**Parameters Description**:
- **bind**: Boolean indicating whether the task should be bound to the task instance (allows access to `self`)
- **name**: Custom name for the task (defaults to module.function_name)
- **base**: Base class for the task
- **retry**: Whether the task should retry on failure
- **max_retries**: Maximum number of retry attempts
- **default_retry_delay**: Default delay between retries in seconds

**Return Value**: Returns a Task object that can be called with `.delay()` or `.apply_async()` methods

**Example 1**: Simple Job Notification Task
```python
from celery import Celery
from django.core.mail import send_mail
from myapp.models import JobPost, Author, User

app = Celery('job_portal')

@app.task
def send_job_notification(job_post_id):
    """Send email notification when a new job is posted"""
    try:
        job_post = JobPost.objects.get(id=job_post_id)
        author = job_post.author
        user = author.user
        
        subject = f"New Job Posted: {job_post.title}"
        message = f"""
        Dear {user.first_name},
        
        Your job posting "{job_post.title}" has been successfully created.
        
        Details:
        - Title: {job_post.title}
        - Description: {job_post.description}
        - Salary: ${job_post.salary:,}
        - Location: {job_post.location.city}, {job_post.location.state}
        - Skills Required: {', '.join([skill.name for skill in job_post.skills.all()])}
        
        Best regards,
        Job Portal Team
        """
        
        result = send_mail(
            subject=subject,
            message=message,
            from_email='noreply@jobportal.com',
            recipient_list=[user.email],
            fail_silently=False
        )
        
        print(f"Email sent successfully to {user.email}")
        return {
            'status': 'success',
            'recipient': user.email,
            'job_title': job_post.title,
            'sent_at': str(datetime.now())
        }
        
    except JobPost.DoesNotExist:
        print(f"JobPost with ID {job_post_id} not found")
        return {'status': 'error', 'message': 'Job post not found'}
    except Exception as e:
        print(f"Error sending notification: {str(e)}")
        return {'status': 'error', 'message': str(e)}

# Usage
result = send_job_notification.delay(1)
print(f"Task ID: {result.id}")
print(f"Task Status: {result.status}")
```

```python
@app.task(bind=True, max_retries=3)
def bulk_update_job_skills(self, job_ids, skill_names):
    """Update skills for multiple job posts"""
    try:
        updated_jobs = []
        skills_objs = Skills.objects.filter(name__in=skill_names)
        
        print(f"Found {skills_objs.count()} skills to update")
        print(f"Skills: {[skill.name for skill in skills_objs]}")
        
        for job_id in job_ids:
            job_post = JobPost.objects.get(id=job_id)
            
            # Clear existing skills and add new ones
            job_post.skills.clear()
            job_post.skills.add(*skills_objs)
            
            updated_jobs.append({
                'job_id': job_id,
                'title': job_post.title,
                'skills_added': [skill.name for skill in skills_objs],
                'total_skills': job_post.skills.count()
            })
            
            print(f"Updated job {job_id}: {job_post.title}")
            
        return {
            'status': 'success',
            'updated_count': len(updated_jobs),
            'jobs': updated_jobs,
            'processed_at': str(datetime.now())
        }
        
    except Exception as exc:
        print(f"Task failed: {str(exc)}")
        raise self.retry(exc=exc, countdown=60, max_retries=3)

# Usage
result = bulk_update_job_skills.delay([1, 2, 3], ['Python', 'Django', 'JavaScript'])
print(f"Bulk update task started: {result.id}")
```

```python
@app.task
def analyze_jobs_by_location(location_ids=None):
    """Analyze job distribution and salary statistics by location"""
    if location_ids:
        locations = Location.objects.filter(id__in=location_ids)
    else:
        locations = Location.objects.all()
    
    analysis_results = []
    
    for location in locations:
        # Get all jobs for this location
        jobs = JobPost.objects.filter(location=location)
        
        if jobs.exists():
            job_data = []
            total_salary = 0
            skill_counts = {}
            
            for job in jobs:
                job_skills = [skill.name for skill in job.skills.all()]
                job_info = {
                    'title': job.title,
                    'salary': job.salary,
                    'skills': job_skills,
                    'author_company': job.author.company,
                    'type': job.type
                }
                job_data.append(job_info)
                total_salary += job.salary
                
                # Count skills
                for skill in job_skills:
                    skill_counts[skill] = skill_counts.get(skill, 0) + 1
            
            avg_salary = total_salary / len(job_data) if job_data else 0
            
            location_analysis = {
                'location': {
                    'city': location.city,
                    'state': location.state,
                    'country': location.country
                },
                'job_count': len(job_data),
                'average_salary': round(avg_salary, 2),
                'salary_range': {
                    'min': min([job['salary'] for job in job_data]),
                    'max': max([job['salary'] for job in job_data])
                },
                'top_skills': sorted(skill_counts.items(), key=lambda x: x[1], reverse=True)[:3],
                'jobs': job_data
            }
            
            analysis_results.append(location_analysis)
            
            print(f"Analyzed {location.city}, {location.state}: {len(job_data)} jobs")
    
    final_result = {
        'total_locations_analyzed': len(analysis_results),
        'locations': analysis_results,
        'analyzed_at': str(datetime.now())
    }
    
    print(f"Location analysis completed for {len(analysis_results)} locations")
    return final_result
```

```python
from django.db.models import Q

@app.task
def find_job_matches_for_skills(skill_names, salary_min=None, location_preferences=None):
    """Find job matches based on skills and preferences"""
    try:
        # Get skill objects
        skills = Skills.objects.filter(name__in=skill_names)
        found_skills = [skill.name for skill in skills]
        missing_skills = [skill for skill in skill_names if skill not in found_skills]
        
        print(f"Searching for jobs with skills: {found_skills}")
        if missing_skills:
            print(f"Skills not found in database: {missing_skills}")
        
        # Build query
        query = Q()
        for skill in skills:
            query |= Q(skills=skill)
        
        jobs = JobPost.objects.filter(query).distinct()
        
        # Apply salary filter
        if salary_min:
            jobs = jobs.filter(salary__gte=salary_min)
            print(f"Filtered by minimum salary: ${salary_min:,}")
        
        # Apply location filter
        if location_preferences:
            location_query = Q()
            for location in location_preferences:
                if ',' in location:
                    city, state = [x.strip() for x in location.split(',')]
                    location_query |= Q(location__city__icontains=city, location__state__icontains=state)
                else:
                    location_query |= Q(location__city__icontains=location) | Q(location__state__icontains=location)
            jobs = jobs.filter(location_query)
            print(f"Filtered by locations: {location_preferences}")
        
        # Analyze matches
        job_matches = []
        for job in jobs:
            job_skills = [skill.name for skill in job.skills.all()]
            matching_skills = [skill for skill in job_skills if skill in skill_names]
            skill_match_percentage = (len(matching_skills) / len(skill_names)) * 100 if skill_names else 0
            
            job_match = {
                'job_id': job.id,
                'title': job.title,
                'description': job.description,
                'salary': job.salary,
                'type': job.type,
                'company': job.author.company,
                'location': {
                    'city': job.location.city,
                    'state': job.location.state,
                    'full_address': f"{job.location.street}, {job.location.city}, {job.location.state}"
                },
                'required_skills': job_skills,
                'matching_skills': matching_skills,
                'skill_match_percentage': round(skill_match_percentage, 2),
                'posted_by': f"{job.author.user.first_name} {job.author.user.last_name}",
                'posted_date': job.date.strftime('%Y-%m-%d'),
                'expiry_date': job.expiry.strftime('%Y-%m-%d') if job.expiry else None
            }
            job_matches.append(job_match)
        
        # Sort by skill match percentage
        job_matches.sort(key=lambda x: x['skill_match_percentage'], reverse=True)
        
        result = {
            'search_criteria': {
                'skills_searched': skill_names,
                'found_skills': found_skills,
                'missing_skills': missing_skills,
                'minimum_salary': salary_min,
                'location_preferences': location_preferences
            },
            'total_matches': len(job_matches),
            'matches': job_matches,
            'search_performed_at': str(datetime.now())
        }
        
        print(f"Found {len(job_matches)} job matches")
        return result
        
    except Exception as e:
        error_msg = f"Error in job matching: {str(e)}"
        print(error_msg)
        return {'status': 'error', 'message': error_msg}

# Usage
```

### 2.2 Task Execution Methods

#### .delay() Method

**Full Description**: The `.delay()` method is the simplest way to execute a Celery task asynchronously. It's a shortcut for `.apply_async()` and immediately queues the task for execution by available workers. This method returns an AsyncResult object that can be used to track the task's status and retrieve results.

**Syntax**:
```python
result = task_function.delay(arg1, arg2, kwarg1=value)
```

**Parameters Description**:
- Arguments are passed directly to the task function
- No additional configuration options available (use apply_async for advanced options)

**Return Value**: AsyncResult object containing task ID and methods to check status/get results

**Example 1**: Simple Task Execution with Job Creation
```python
@app.task
def create_job_post_async(title, description, salary, author_id, location_id, skill_ids):
    """Create a new job post asynchronously"""
    try:
        author = Author.objects.get(id=author_id)
        location = Location.objects.get(id=location_id)
        skills = Skills.objects.filter(id__in=skill_ids)
        
        # Create the job post
        job_post = JobPost.objects.create(
            title=title,
            description=description,
```

### 2.3 Advanced Task Execution with .apply_async()

#### .apply_async() Method

**Full Description**: The `.apply_async()` method provides full control over task execution with advanced options like countdown delays, ETA scheduling, routing, callbacks, and custom serialization. It's the most flexible way to execute Celery tasks and is used when you need more than basic asynchronous execution.

**Syntax**:
```python
result = task_function.apply_async(
    args=(arg1, arg2),
    kwargs={'kwarg1': value1},
    countdown=60,
    eta=datetime_obj,
    expires=3600,
    retry=True,
    retry_policy={'max_retries': 3, 'interval_start': 0, 'interval_step': 0.2},
    routing_key='queue_name',
    priority=5
)
```

**Parameters Description**:
- **args**: Tuple of positional arguments for the task
- **kwargs**: Dictionary of keyword arguments for the task
- **countdown**: Delay execution by specified seconds
- **eta**: Execute at specific datetime
- **expires**: Task expires after specified seconds
- **retry**: Enable automatic retry on failure
- **retry_policy**: Dictionary defining retry behavior
- **routing_key**: Route task to specific queue
- **priority**: Task priority (0-9, higher is more priority)

**Return Value**: AsyncResult object with enhanced tracking capabilities

**Example 1**: Scheduled Job Expiry Task
```python
from datetime import datetime, timedelta

@app.task(bind=True)
def expire_old_jobs(self, days_threshold=30):
    """Expire job posts older than specified days"""
    try:
        cutoff_date = datetime.now() - timedelta(days=days_threshold)
        
        # Find jobs to expire
        jobs_to_expire = JobPost.objects.filter(
            date__lt=cutoff_date,
            expiry__isnull=True  # Only jobs without expiry set
        )
        
        expired_jobs = []
        for job in jobs_to_expire:
            # Set expiry to today
            job.expiry = datetime.now().date()
            job.save()
            
            expired_job_info = {
                'job_id': job.id,
                'title': job.title,
                'posted_date': job.date.strftime('%Y-%m-%d'),
                'author': f"{job.author.user.first_name} {job.author.user.last_name}",
                'company': job.author.company,
                'expired_on': job.expiry.strftime('%Y-%m-%d')
            }
            expired_jobs.append(expired_job_info)
            
            print(f"Expired job: {job.title} (ID: {job.id})")
        
        result = {
            'total_expired': len(expired_jobs),
            'threshold_days': days_threshold,
            'cutoff_date': cutoff_date.strftime('%Y-%m-%d'),
            'expired_jobs': expired_jobs,
            'processed_at': str(datetime.now())
        }
        
        print(f"Expired {len(expired_jobs)} job posts older than {days_threshold} days")
        return result
        
    except Exception as exc:

        raise self.retry(exc=exc, countdown=300, max_retries=3)
```

### 2.4 Periodic Tasks with Celery Beat

#### @periodic_task Decorator

**Full Description**: The `@periodic_task` decorator (now deprecated in favor of beat_schedule) allows you to define tasks that run automatically at specified intervals. This is essential for maintenance tasks, regular data processing, and scheduled operations in Django applications.

**Syntax**:
```python
from celery.schedules import crontab

app.conf.beat_schedule = {
    'task-name': {
        'task': 'myapp.tasks.task_function',
        'schedule': crontab(minute=0, hour=4),  # Run daily at 4 AM
        'args': (arg1, arg2),
        'kwargs': {'kwarg1': 'value1'}
    }
}
```

**Parameters Description**:
- **task**: Full path to the task function
- **schedule**: Schedule object (crontab, interval, etc.)
- **args**: Tuple of positional arguments
- **kwargs**: Dictionary of keyword arguments
- **options**: Additional task options (routing, expires, etc.)

**Return Value**: Scheduled execution based on defined intervals

**Example 1**: Daily Job Statistics Report

### 2.5 Task Result Handling

#### AsyncResult Object Methods

**Full Description**: AsyncResult objects provide methods to interact with task execution results, check status, and handle task lifecycle events. These objects are returned when tasks are executed and offer comprehensive task management capabilities.

**Syntax**:
```python
result = task_function.delay(args)
status = result.status
task_result = result.get(timeout=10)
result.forget()
result.revoke(terminate=True)
```

**Parameters Description**:
- **timeout**: Maximum time to wait for result (in seconds)
- **propagate**: Whether to raise exceptions from task
- **interval**: Polling interval for result checking
- **terminate**: Whether to terminate running task when revoking

**Return Value**: Various depending on method (status string, task result, boolean)

```python
# Using a string here means the worker doesn't have to serialize

app.config_from_object('django.conf:settings', namespace='CELERY')

# Autodiscover tasks from all registered Django apps
app.autodiscover_tasks()
```

```python
@shared_task(bind=True, max_retries=3)
def send_job_notification(self, job_post_id, notification_type='created'):
    """
    Send email notifications for job-related events
    
    Args:
        job_post_id (int): ID of the job post
        notification_type (str): Type of notification ('created', 'updated', 'expired')
    """
    try:
        job_post = JobPost.objects.select_related('author__user', 'location').prefetch_related('skills').get(id=job_post_id)
        
        notification_templates = {
            'created': {
                'subject': f"Job Posted Successfully: {job_post.title}",
                'message': f"""
Dear {job_post.author.user.first_name},

Your job posting has been successfully created and is now live on our platform.

Job Details:
- Title: {job_post.title}
- Company: {job_post.author.company}
- Location: {job_post.location.city}, {job_post.location.state}
- Salary: ${job_post.salary:,}
- Type: {job_post.type}
- Skills Required: {', '.join([skill.name for skill in job_post.skills.all()])}

Your job post is now visible to potential candidates.

Best regards,
Job Portal Team
                """
            },
            'updated': {
                'subject': f"Job Updated: {job_post.title}",
                'message': f"Your job posting '{job_post.title}' has been successfully updated."
            },
            'expired': {
                'subject': f"Job Expired: {job_post.title}",
                'message': f"Your job posting '{job_post.title}' has expired and is no longer visible to candidates."
            }
        }
        
        template = notification_templates.get(notification_type, notification_templates['created'])
        
        result = send_mail(
            subject=template['subject'],
            message=template['message'],
            from_email='noreply@jobportal.com',
            recipient_list=[job_post.author.user.email],
            fail_silently=False
        )
        
        logger.info(f"Notification sent successfully to {job_post.author.user.email}")
        
        return {
            'status': 'success',
            'notification_type': notification_type,
            'recipient': job_post.author.user.email,
            'job_title': job_post.title,
            'sent_at': str(datetime.now())
        }
        
    except JobPost.DoesNotExist:
        logger.error(f"JobPost with ID {job_post_id} not found")
        return {'status': 'error', 'message': 'Job post not found'}
    
    except Exception as exc:
        logger.error(f"Failed to send notification: {str(exc)}")
        raise self.retry(exc=exc, countdown=60 * (self.request.retries + 1))

@shared_task(bind=True)
def bulk_update_job_skills(self, job_ids, skill_names, operation='replace'):
    """
    Bulk update skills for multiple job posts
    
    Args:
        job_ids (list): List of job post IDs
        skill_names (list): List of skill names
        operation (str): 'replace', 'add', or 'remove'
    """
    try:
        jobs = JobPost.objects.filter(id__in=job_ids)
        skills = Skills.objects.filter(name__in=skill_names)
        
        updated_jobs = []
        
        for job in jobs:
            if operation == 'replace':
                job.skills.clear()
                job.skills.add(*skills)
            elif operation == 'add':
                job.skills.add(*skills)
            elif operation == 'remove':
                job.skills.remove(*skills)
            
            updated_jobs.append({
                'job_id': job.id,
                'title': job.title,
                'operation': operation,
                'skills_affected': skill_names,
                'current_skills': [skill.name for skill in job.skills.all()]
            })
        
        return {
            'status': 'success',
            'operation': operation,
            'jobs_updated': len(updated_jobs),
            'jobs': updated_jobs,
            'processed_at': str(datetime.now())
        }
        
    except Exception as exc:
        logger.error(f"Bulk skill update failed: {str(exc)}")
        raise self.retry(exc=exc, countdown=60)


@shared_task
def create_job_post_async(job_data):

    Create a new job post asynchronously
    
    Args:
        job_data (dict): Dictionary containing job post information
    """

        skills = Skills.objects.filter(id__in=job_data['skill_ids'])

            'status': 'success',
            'job_id': job_post.id,
            'title': job_post.title,

            'created_at': job_post.date.strftime('%Y-%m-%d %H:%M:%S'),

        }
        
    except Exception as e:

@shared_task
def analyze_jobs_by_location(location_ids=None, detailed=True):
    """Comprehensive location-based job analysis"""
    if location_ids:
        locations = Location.objects.filter(id__in=location_ids)
    else:
        locations = Location.objects.all()
    
    analysis_results = []
    
    for location in locations:
        jobs = JobPost.objects.filter(location=location).select_related('author__user').prefetch_related('skills')
        
        if jobs.exists():
            # Basic statistics
            job_count = jobs.count()
            salaries = [job.salary for job in jobs]
            avg_salary = sum(salaries) / len(salaries)
            
            # Skill analysis for this location
            skill_counts = {}
            company_counts = {}
            type_counts = {'Full Time': 0, 'Part Time': 0}
            
            job_details = []
            for job in jobs:
                # Skill counting
                for skill in job.skills.all():
                    skill_counts[skill.name] = skill_counts.get(skill.name, 0) + 1
                
                # Company counting
                company_counts[job.author.company] = company_counts.get(job.author.company, 0) + 1
                
                # Type counting
                type_counts[job.type] += 1
                
                if detailed:
                    job_details.append({
                        'id': job.id,
                        'title': job.title,
                        'company': job.author.company,
                        'salary': job.salary,
                        'type': job.type,
                        'skills': [skill.name for skill in job.skills.all()],
                        'posted_date': job.date.strftime('%Y-%m-%d')
                    })
            
            location_analysis = {
                'location_info': {
                    'id': location.id,
                    'city': location.city,
                    'state': location.state,
                    'country': location.country,
                    'full_address': f"{location.street}, {location.city}, {location.state} {location.zip}"
                },
                'job_statistics': {
                    'total_jobs': job_count,
                    'average_salary': round(avg_salary, 2),
                    'salary_range': {
                        'min': min(salaries),
                        'max': max(salaries)
                    },
                    'job_types': type_counts
                },
                'skill_demand': sorted(skill_counts.items(), key=lambda x: x[1], reverse=True)[:5],
                'top_companies': sorted(company_counts.items(), key=lambda x: x[1], reverse=True)[:5]
            }
            
            if detailed:
                location_analysis['job_details'] = job_details
            
            analysis_results.append(location_analysis)
    
    return {
        'total_locations_analyzed': len(analysis_results),
        'locations': analysis_results,
        'analysis_completed_at': str(datetime.now())
    }

@shared_task
def find_job_matches_for_skills(skill_names, filters=None):
    """
    Advanced job matching algorithm based on skills and filters
    
    Args:
        skill_names (list): List of skill names to match
        filters (dict): Additional filters (salary_min, location, job_type)
    """
    filters = filters or {}
    
    try:
        # Get skill objects
        skills = Skills.objects.filter(name__in=skill_names)
        found_skills = [skill.name for skill in skills]
        
        # Base query for jobs with any matching skills
        jobs_query = JobPost.objects.filter(skills__in=skills).distinct()
        
        # Apply filters
        if filters.get('salary_min'):
            jobs_query = jobs_query.filter(salary__gte=filters['salary_min'])
        
        if filters.get('job_type'):
            jobs_query = jobs_query.filter(type=filters['job_type'])
        
        if filters.get('location_city'):
            jobs_query = jobs_query.filter(location__city__icontains=filters['location_city'])
        
        if filters.get('location_state'):
            jobs_query = jobs_query.filter(location__state__icontains=filters['location_state'])
        
        # Only active jobs
        today = datetime.now().date()
        jobs_query = jobs_query.filter(Q(expiry__isnull=True) | Q(expiry__gte=today))
        
        jobs = jobs_query.select_related('author__user', 'location').prefetch_related('skills')
        
        # Calculate match scores
        job_matches = []
        for job in jobs:
            job_skills = [skill.name for skill in job.skills.all()]
            matching_skills = [skill for skill in job_skills if skill in skill_names]
            
            # Calculate different scoring metrics
            skill_coverage = len(matching_skills) / len(skill_names) * 100 if skill_names else 0
            requirement_coverage = len(matching_skills) / len(job_skills) * 100 if job_skills else 0
            
            # Combined score (weighted average)
            match_score = (skill_coverage * 0.7 + requirement_coverage * 0.3)
            
            job_match = {
                'job_id': job.id,
                'title': job.title,
                'description': job.description,
                'company': job.author.company,
                'salary': job.salary,
                'type': job.type,
                'location': {
                    'city': job.location.city,
                    'state': job.location.state,
                    'address': f"{job.location.city}, {job.location.state}"
                },
                'required_skills': job_skills,
                'matching_skills': matching_skills,
                'missing_skills': [skill for skill in job_skills if skill not in skill_names],
                'match_metrics': {
                    'skill_coverage_percent': round(skill_coverage, 2),
                    'requirement_coverage_percent': round(requirement_coverage, 2),
                    'overall_match_score': round(match_score, 2)
                },
                'contact_info': {
                    'recruiter': f"{job.author.user.first_name} {job.author.user.last_name}",
                    'designation': job.author.designation
                },
                'posted_date': job.date.strftime('%Y-%m-%d'),
                'expires_on': job.expiry.strftime('%Y-%m-%d') if job.expiry else 'No expiry'
            }
            job_matches.append(job_match)
        
        # Sort by match score
        job_matches.sort(key=lambda x: x['match_metrics']['overall_match_score'], reverse=True)
        
        result = {
            'search_criteria': {
                'skills_requested': skill_names,
                'skills_found': found_skills,
                'filters_applied': filters
            },
            'match_summary': {
                'total_jobs_found': len(job_matches),
                'perfect_matches': len([job for job in job_matches if job['match_metrics']['skill_coverage_percent'] == 100]),
                'partial_matches': len([job for job in job_matches if 50 <= job['match_metrics']['skill_coverage_percent'] < 100]),
                'weak_matches': len([job for job in job_matches if job['match_metrics']['skill_coverage_percent'] < 50])
            },
            'job_matches': job_matches,
            'search_performed_at': str(datetime.now())
        }
        
        logger.info(f"Job matching completed: {len(job_matches)} matches found for skills {skill_names}")
        return result
        
    except Exception as e:
        logger.error(f"Job matching failed: {str(e)}")
        return {'status': 'error', 'message': str(e)}

@shared_task
def automated_job_matching():
    """Automated job matching for all active users (hourly task)"""
    # This would typically fetch user preferences from a UserProfile model
    # For demonstration, we'll use our existing authors as "users"
    
    authors = Author.objects.select_related('user').all()
    matching_results = []
    
    for author in authors:
        # Get jobs posted by this author to understand their skill preferences
        author_jobs = JobPost.objects.filter(author=author).prefetch_related('skills')
        
        if author_jobs.exists():
            # Extract skills from author's job postings
            author_skills = set()
            for job in author_jobs:
                author_skills.update([skill.name for skill in job.skills.all()])
            
            # Find matching jobs (excluding their own)
            matches = find_job_matches_for_skills.apply_async(
                args=[list(author_skills)],
                kwargs={'filters': {'salary_min': 50000}}
            ).get()
            
            # Filter out their own jobs
            filtered_matches = [
                match for match in matches.get('job_matches', [])
                if match['job_id'] not in [job.id for job in author_jobs]
            ]
            
            if filtered_matches:
                matching_results.append({
                    'user_id': author.user.id,
                    'user_name': f"{author.user.first_name} {author.user.last_name}",
                    'user_email': author.user.email,
                    'skills_profile': list(author_skills),
                    'matches_found': len(filtered_matches),
                    'top_matches': filtered_matches[:3]  # Top 3 matches
                })
    
    # Send notifications for users with good matches
    for result in matching_results:
        if result['matches_found'] > 0:
            send_job_match_notification.delay(
                result['user_id'],
                result['top_matches']
            )
    
    return {
        'users_processed': len(authors),
        'users_with_matches': len(matching_results),
        'total_matches_found': sum([r['matches_found'] for r in matching_results]),
        'processed_at': str(datetime.now())
    }

@shared_task
def send_job_match_notification(user_id, job_matches):
    """Send job match notification to user"""
    try:
        user = User.objects.get(id=user_id)
        
        if not job_matches:
            return {'status': 'no_matches', 'user': user.email}
        
        # Create email content
        subject = f"New Job Matches Found - {len(job_matches)} Opportunities"
        
        message_content = f"""Dear {user.first_name},

We found {len(job_matches)} new job opportunities that match your skills!

Top Matches:
"""
        
        for i, match in enumerate(job_matches[:3], 1):
            message_content += f"""
{i}. {match['title']} at {match['company']}
   Salary: ${match['salary']:,}
   Location: {match['location']['address']}
   Match Score: {match['match_metrics']['overall_match_score']:.1f}%
   Skills: {', '.join(match['matching_skills'])}
"""
        
        message_content += f"""

Visit our platform to view all matches and apply.

Best regards,
Job Portal Team

This email was sent on {datetime.now().strftime('%Y-%m-%d %H:%M:%S')} UTC.
"""
        
        send_mail(
            subject=subject,
            message=message_content,
            from_email='noreply@jobportal.com',
            recipient_list=[user.email],
            fail_silently=False
        )
        
        return {
            'status': 'success',
            'user': user.email,
            'matches_sent': len(job_matches),
            'sent_at': str(datetime.now())
        }
        
    except Exception as e:
        logger.error(f"Failed to send job match notification: {str(e)}")
        return {'status': 'error', 'message': str(e)}

@shared_task
def weekly_database_cleanup():
    """Comprehensive weekly database maintenance"""
    cleanup_report = {
        'cleanup_date': datetime.now().strftime('%Y-%m-%d'),
        'operations_performed': []
    }
    
    # 1. Remove expired jobs older than 30 days
    expired_cutoff = datetime.now().date() - timedelta(days=30)
    old_expired_jobs = JobPost.objects.filter(expiry__lt=expired_cutoff)
    
    expired_job_count = old_expired_jobs.count()
    if expired_job_count > 0:
        expired_jobs_data = []
        for job in old_expired_jobs:
            expired_jobs_data.append({
                'id': job.id,
                'title': job.title,
                'company': job.author.company,
                'expired_date': job.expiry.strftime('%Y-%m-%d')
            })
        
        old_expired_jobs.delete()
        
        cleanup_report['operations_performed'].append({
            'operation': 'removed_old_expired_jobs',
            'count': expired_job_count,
            'details': expired_jobs_data
        })
    
    # 2. Identify unused skills
    unused_skills = Skills.objects.annotate(
        job_count=Count('typeskill')
    ).filter(job_count=0)
    
    cleanup_report['operations_performed'].append({
        'operation': 'identified_unused_skills',
        'count': unused_skills.count(),
        'details': [skill.name for skill in unused_skills]
    })
    
    # 3. Identify unused locations
    unused_locations = Location.objects.annotate(
        job_count=Count('areaname')
    ).filter(job_count=0)
    
    cleanup_report['operations_performed'].append({
        'operation': 'identified_unused_locations',
        'count': unused_locations.count(),
        'details': [f"{loc.city}, {loc.state}" for loc in unused_locations]
    })
    
    # 4. Identify inactive authors (no jobs in last 90 days)
    inactive_cutoff = datetime.now().date() - timedelta(days=90)
    inactive_authors = Author.objects.annotate(
        recent_job_count=Count('personname', filter=Q(personname__date__gte=inactive_cutoff))
    ).filter(recent_job_count=0)
    
    cleanup_report['operations_performed'].append({
        'operation': 'identified_inactive_authors',
        'count': inactive_authors.count(),
        'details': [f"{author.user.first_name} {author.user.last_name}" for author in inactive_authors]
    })
    
    # 5. Update job statistics
    total_stats = {
        'total_jobs': JobPost.objects.count(),
        'active_jobs': JobPost.objects.filter(Q(expiry__isnull=True) | Q(expiry__gte=datetime.now().date())).count(),
        'total_skills': Skills.objects.count(),
        'total_locations': Location.objects.count(),
        'total_authors': Author.objects.count()
    }
    
    cleanup_report['current_statistics'] = total_stats
    cleanup_report['completed_at'] = str(datetime.now())

@shared_task
def send_daily_digest():
    """Send daily digest emails to all users"""
    today = datetime.now().date()
    
    # Get today's new jobs
    new_jobs = JobPost.objects.filter(
        date__date=today
    ).select_related('author__user', 'location').prefetch_related('skills')
    
    if not new_jobs.exists():
        return {'status': 'no_new_jobs', 'date': today.strftime('%Y-%m-%d')}
    
    # Get all users
    users = User.objects.all()
    
    digest_results = []
    
    for user in users:
        subject = f"Daily Job Digest - {new_jobs.count()} New Jobs Today"
        
        message = f"""Dear {user.first_name},

Here are the {new_jobs.count()} new job opportunities posted today:

"""
        
        for job in new_jobs:
            message += f"""
• {job.title} at {job.author.company}
  Salary: ${job.salary:,} | Location: {job.location.city}, {job.location.state}
  Skills: {', '.join([skill.name for skill in job.skills.all()])}
  
"""
        
        message += f"""

Visit our platform to view detailed job descriptions and apply.

Best regards,
Job Portal Team

Daily Digest for {today.strftime('%B %d, %Y')}
"""
        
        try:
            send_mail(
                subject=subject,
                message=message,
                from_email='digest@jobportal.com',
                recipient_list=[user.email],

                'status': 'failed',
                'error': str(e)
            })
    
    return {
        'digest_date': today.strftime('%Y-%m-%d'),
        'new_jobs_count': new_jobs.count(),
        'emails_attempted': len(users),
        'emails_sent': len([r for r in digest_results if r['status'] == 'sent']),
```

```python
@shared_task
def send_job_alert_idempotent(user_id, job_id, alert_key=None):
    """Idempotent job alert - can be run multiple times safely"""
    if not alert_key:
        alert_key = f"alert_{user_id}_{job_id}_{datetime.now().date()}"

    # Send alert logic here
```

```python
@shared_task(bind=True, autoretry_for=(Exception,), retry_kwargs={'max_retries': 3, 'countdown': 60})
def robust_job_processing(self, job_id):
    """Task with comprehensive retry strategy"""
    try:
        # Task logic here
        job = JobPost.objects.get(id=job_id)
        # Process job...
        return {'status': 'success', 'job_id': job_id}
        
    except JobPost.DoesNotExist:
        # Don't retry for this error
        logger.error(f"Job {job_id} not found")
        return {'status': 'error', 'message': 'Job not found'}
        
    except ConnectionError as exc:
        # Custom retry for connection errors
        logger.warning(f"Connection error, retrying: {exc}")
        raise self.retry(exc=exc, countdown=120, max_retries=5)
        
    except Exception as exc:
        # General exception handling
        logger.error(f"Task failed: {exc}")
        if self.request.retries < 3:
            raise self.retry(exc=exc, countdown=60 * (self.request.retries + 1))
        else:
            # Send failure notification after max retries
            send_admin_notification.delay(f"Task failed permanently: {exc}")
            raise
```

```python
# monitoring/tasks.py

@shared_task
def monitor_task_queues():
    """Monitor task queue health"""
    from celery import current_app
    
    inspect = current_app.control.inspect()
    
    # Get queue lengths
    active_tasks = inspect.active()
    scheduled_tasks = inspect.scheduled()
    reserved_tasks = inspect.reserved()
    
    # Calculate metrics
    total_active = sum(len(tasks) for tasks in (active_tasks or {}).values())
    total_scheduled = sum(len(tasks) for tasks in (scheduled_tasks or {}).values())
    
    # Alert thresholds
    if total_active > 100:
        send_admin_alert.delay(
            subject="High Task Queue Load",
            message=f"Active tasks: {total_active}, Scheduled: {total_scheduled}"
        )
    
    return {
        'active_tasks': total_active,
        'scheduled_tasks': total_scheduled,
        'worker_status': 'healthy' if total_active < 50 else 'overloaded'
    }
```
