# Everyday Most Used Services for Django/AWS/Docker Developer

## 🐳 Containers (Daily Use)
- Amazon ECS          → Run Docker containers
- Amazon ECR          → Store Docker images
- AWS Fargate         → Serverless container execution

## 💾 Storage (Daily Use)
- Amazon S3           → Static files, media uploads, backups

## 🗄️ Database (Daily Use)
- Amazon RDS          → PostgreSQL/MySQL for Django
- Amazon ElastiCache  → Redis for caching & Celery broker

## 🌐 Networking (Daily Use)
- Amazon VPC          → Network isolation
- Elastic Load Balancing (ELB) → Distribute traffic
- Route 53            → DNS management

## 🔒 Security (Daily Use)
- AWS IAM             → Access management
- AWS Secrets Manager → Store DB passwords, API keys
- AWS Certificate Manager → SSL/TLS certificates

## 🔗 Application Integration (Daily Use)
- Amazon SQS          → Message queue for Celery tasks
- Amazon SNS          → Notifications
- Amazon API Gateway  → REST API management

## 📊 Monitoring (Daily Use)
- AWS CloudWatch      → Logs, metrics, alarms
- AWS X-Ray           → Trace requests (debugging)

## 🛠️ CI/CD (Daily Use)
- AWS CodePipeline    → Automated deployments
- AWS CodeBuild       → Build Docker images
- AWS CodeDeploy      → Deploy to ECS/EC2

## ⚙️ Infrastructure as Code
- AWS CloudFormation  → Infrastructure templates (or use Terraform)

---
